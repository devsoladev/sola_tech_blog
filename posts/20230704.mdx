## Routing

애플리케이션 내의 라우팅은 사용자가 요청한 URL 또는 이벤트를 해석하여 해당 화면으로 전환하는 네비게이션을 관리하기 위한 기능을 의미한다.

## React Router

### Route Component

사용자의 브라우저 주소 경로에 따라 원하는 컴포넌트를 렌더링하기 위해 사용한다.

```javascript
<Route path="주소" element={렌더링할 컴포넌트}>
```

### Link Component

다른 페이지로 이동하는 링크를 노출하기 위해 사용한다.

```javascript
<Link to='경로'>링크</Link>
```
