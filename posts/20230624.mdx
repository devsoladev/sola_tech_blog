## About React

React는 사용자 인터페이스를 만들기 위한 JavaScript 라이브러리 중 하나로, Meta에서 개발한 오픈소스 프로젝트이다.<br />
React의 대표적인 특징은 다음과 같다.<br />

- React의 특징
  - 가상 DOM 지원
  - Single Page Application
  - 단방향 데이터 바인딩
  - 선언형 프로그래밍
  - 컴포넌트 기반

## 너도 나도 React를 사용하는 이유

React는 전세계적으로 높은 점유율을 차지하고 있다. 2022년 Stack Overflow에서 개발자를 대상으로 실시한 설문 조사에 따르면 웹 프레임워크 및 기술 파트에서 React가 Node.js 다음으로 많은 응답률을 보였다.

![Stack Overflow Survey 2022](/20230626/stackoverflowsurvey.jpg)

### Reuseable

애플리케이션의 규모가 커질수록 코드의 재사용성이 매우 중요해진다. 리액트는 독립적인 코드 조각이라 할 수 있는 컴포넌트를 기반으로 한다. 이 컴포넌트를 조립하는 방식으로 UI를 구성하고, 반복되는 코드를 컴포넌트화 하여 코드의 중복을 줄일 수 있다.

### Ecosystem

오늘날 리액트 생태계는 매우 활발하다. 여느 기술보다 다양한 커뮤니티 활동이 이루어지고 있으며, 수많은 참고 자료와 강의들이 제공되고 있기 때문에 학습하기에도 용이하다. 최근 리액트 18버전이 업데이트 되었는데, 기존의 불편함을 보완한 것 뿐만 아니라, 새로운 기능들이 대거 추가되었다. 대표적으로 Automatic Batching, Transition, Suspense 등이 있다.

### Posibility

오늘날 다양한 React 프레임워크나 라이브러리가 등장하면서 React로 만들 수 있는 분야가 점차 확장되고 있다. Next.js를 통해 Sever Side Rendering을 구현할 수 있다. 또한 모바일 애플리케이션을 제작할 수 있는 React Native가 있기 때문에 React 하나를 배우면 웹/앱을 모두 다룰 수 있다. 심지어 최근에는 VR 애플리케이션을 만들 수 있는 React 360도 개발되었다. 그밖에도 Remix, React Query, Recoil 등 React 프레임워크나 라이브러리들이 지속적으로 나오고 있다. 이렇듯 React 생태계는 계속 진화하는 중이다.
